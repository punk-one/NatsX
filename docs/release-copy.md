## GitHub Release Body

### Chinese

`NatsX 1.0.1` 是一个面向 `NATS / JetStream` 的 Windows 桌面客户端，基于 `Go + Wails + React + Ant Design` 构建，聚焦连接管理、消息收发、`Request / Reply` 调试、日志查看与 JetStream 运维。

本次版本包含：

- 连接创建、编辑、删除、分组与本地持久化
- `No Auth`、`Username / Password`、`Token`、`TLS / mTLS`、`NKey`、`Credentials`
- Subject 订阅、消息收发与多种载荷视图
- `Request / Reply` 调试、重发与结果对比
- 日志查看、消息历史、Headers 与 Payload 检查
- JetStream Stream / Consumer 管理与 `Ack / Nak / Term`
- 基于纯 Go `SQLite` 的设置、连接、升级状态与日志持久化
- 中英文界面切换，并保存当前语言配置

项目地址：

- `https://github.com/punk-one/NatsX`

### English

`NatsX 1.0.1` is a Windows desktop client for `NATS / JetStream`, built with `Go + Wails + React + Ant Design`, focusing on connection management, messaging, `Request / Reply`, logging, and JetStream operations.

Highlights in this release:

- Connection create, edit, delete, grouping, and persistence
- `No Auth`, `Username / Password`, `Token`, `TLS / mTLS`, `NKey`, and `Credentials`
- Subject subscribe flow, publish and receive workflow, and multiple payload views
- `Request / Reply` debugging, replay, and comparison
- Logs, message history, headers, and payload inspection
- JetStream Stream / Consumer management with `Ack / Nak / Term`
- Pure-Go `SQLite` persistence for settings, connections, update state, and logs
- Chinese and English UI support with saved language preference
- Portable Windows `zip` distribution for direct download and use

Project links:

- `https://github.com/punk-one/NatsX`
