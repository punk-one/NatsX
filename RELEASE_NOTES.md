# Release Notes

## NatsX 1.0.1

Release date: `2026-03-23`

`NatsX 1.0.1` is the first polished Windows desktop release for `NATS / JetStream`, focused on local productivity, connection reuse, payload inspection, and reliable persistence.

### Highlights

- Windows desktop client built with `Go + Wails + React + Ant Design`
- Core `NATS` and `JetStream` workflows in a single desktop workspace
- Reworked local persistence using pure-Go `SQLite` without CGO dependencies
- Chinese and English UI support with persisted language preference
- Built-in update checking with GitHub Release awareness and manual upgrade flow

### What is available in 1.0.1

- Connection management with create, edit, delete, grouping, import, and export
- Authentication support for `No Auth`, `Username / Password`, `Token`, `TLS / mTLS`, `NKey`, and `Credentials`
- Managed local resource storage for uploaded credentials and TLS files
- Publish, subscribe, reply, republish, inspect, and format payloads with headers
- `Request / Reply` replay and side-by-side compare tooling
- Viewer, payload inspector, message history, and log inspection workflows
- JetStream Stream and Consumer operations, including pull fetch and `Ack / Nak / Term`
- Persistent settings, connections, update state, and runtime logs stored in `database/natsx.db`

### Packaging and delivery notes

- Windows desktop is the primary release target for `1.0.1`
- Release artifacts are generated by `scripts/release-windows.ps1`
- The recommended public distribution format for `1.0.1` is the portable `zip` package
- The release package includes both English and Simplified Chinese README files
- The preview image used by the README is bundled as `docs/screenshot.png`

### Follow-up areas
