# Release Notes

## NatsX 1.0.2

Release date: `2026-03-28`

`NatsX 1.0.2` is a cross-platform desktop release for `NATS / JetStream`, focused on stable packaging, structured upgrade metadata, and a smoother manual upgrade workflow.

### Highlights

- Desktop client built with `Go + Wails + React + Ant Design`
- Public release packages for both `windows-amd64` and `linux-amd64`
- Structured release manifest via `latest.json` for future updater compatibility
- Combined checksum distribution via `SHA256SUMS`
- Pure-Go `SQLite` persistence for settings, connections, update state, and logs

### Included in this release

- Windows portable package: `NatsX-1.0.2-windows-amd64.zip`
- Linux portable package: `NatsX-1.0.2-linux-amd64.tar.gz`
- Per-package checksum files: `*.sha256.txt`
- Combined checksum file: `SHA256SUMS`
- Release helper documents and preview image

### What is available in 1.0.2

- Connection management with create, edit, delete, grouping, import, and export
- Authentication support for `No Auth`, `Username / Password`, `Token`, `TLS / mTLS`, `NKey`, and `Credentials`
- Managed local resource storage for uploaded credentials and TLS files
- Publish, subscribe, reply, republish, inspect, and format payloads with headers
- `Request / Reply` replay and side-by-side compare tooling
- Viewer, payload inspector, message history, and log inspection workflows
- JetStream Stream and Consumer operations, including pull fetch and `Ack / Nak / Term`
- Persistent settings, connections, update state, and runtime logs stored in `database/natsx.db`

### Packaging and delivery notes

- Windows artifacts are generated by `scripts/release-windows.ps1`
- Linux artifacts are generated by `scripts/release-linux.sh`
- The latest release can include a machine-readable `latest.json` asset
- The preview image used by the README is bundled as `docs/screenshot.png`

### Follow-up areas

- Optional installer packaging for broader Windows distribution
- Signature and checksum verification in the in-app updater flow
- More guided troubleshooting material for deep inspection workflows
